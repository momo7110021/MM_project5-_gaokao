# 交互式页面使用说明

> 高考志愿智能推荐系统 — Streamlit UI
>
> 版本: v3.1.2 UI

---

## 运行方式

```bash
# 1. 安装依赖
pip install -r requirements.txt

# 2. 启动交互式页面
streamlit run streamlit_app.py

# 3. 浏览器自动打开 http://localhost:8501
```

---

## 页面功能

### 左侧输入栏

| 分区 | 输入项 | 说明 |
|------|--------|------|
| 考生信息 | 省份、年份、科类、分数、位次 | 必填 |
| 新高考选科 | 3+1+2/3+3 模式选择 + 六选三科目组合 |
| 方案偏好 | 激进型/均衡型/保守型 | 决定冲稳保垫比例和权重 |
| 家庭约束 | 预算、是否接受调剂/中外合作/远程 | 硬约束过滤 |
| 偏好城市 | 多选 | 影响城市评分 |
| 排斥专业 | 多选 + 手动输入 | 硬约束过滤 |

### 选科输入说明

- **3+3 模式**: 从物理/化学/生物/历史/地理/政治中任选 3 门
- **3+1+2 模式**: 首选科目(物理/历史二选一) + 再选科目(化学/生物/地理/政治四选二)。再选科目固定为化学、生物、地理、政治，不可选物理/历史
- 系统自动推导兼容的 `subject_type`(物理类/历史类/综合类)传给模型

### 数据模式

| 模式 | 说明 |
|------|------|
| 模拟数据 Demo | 使用 data_generator.py 生成的模拟数据，适合演示和测试 |
| 广东样例数据 | 加载 data/ 下广东省2021-2026年真实一分一段表和投档线，仅支持广东省物理类/历史类 |
| 真实数据模式 | 需手动配置 data/processed 目录，适合正式部署 |

### 结果页（6 个 Tab）

| Tab | 内容 |
|-----|------|
| 推荐总览 | 考生概览卡片 + 冲稳保垫柱状图 |
| 志愿表 | 推荐志愿表（支持按等级/风险筛选、关键词搜索、点击查看详情） |
| 风险分析 | 六类风险评分进度条 + 十大问题列表 + 咨询师复核清单 |
| 专业就业 | 推荐方案中主要专业的就业评分排序表 |
| 解释报告 | 家长端通俗解释，含风险提示和"不构成录取保证"说明 |
| 数据说明 | 数据来源说明、模拟数据免责声明、JSON 元信息 |

### 导出功能

页面底部提供三个下载按钮：

1. **JSON** — 完整模型输出（含所有字段）
2. **Markdown 报告** — 中文业务报告
3. **CSV 志愿表** — 志愿表表格

## 注意事项

- 当前版本使用**模拟数据**，仅供教学和系统原型展示
- 页面顶部和 Tab6 醒目标注了免责声明
- 模型输出字段与论文、README、docs/api_spec.json 完全一致
- 所有计算使用现有 `src/pipeline.py` 和 `src/data_generator.py`，未重新实现模型逻辑
